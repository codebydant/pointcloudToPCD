# pointcloudToPCD
C++ application to convert ply file, txt file or xyz file point cloud to PCD file

-------------------
## Compile
* Create a "build" folder

in the main folder:

	- cd build  
	- cmake ../src/
  - make
       
        	 
### Test

	  ./pcl-visualizer <pcd file> 
    ./pcl-visualizer <ply file> 
  	./pcl-visualizer <txt file> 
  	./pcl-visualizer <xyz file>
